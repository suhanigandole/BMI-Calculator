# BMI Calculator

## Description
A simple Python application that calculates Body Mass Index (BMI) based on a user's weight and height.

## Features
- Takes weight in kilograms
- Takes height in meters
- Calculates BMI
- Classifies BMI as:
  - Underweight
  - Normal Weight
  - Overweight
  - Obese
- Input validation for invalid values

## BMI Formula

BMI = Weight (kg) / Height² (m²)

## How to Run

1. Install Python 3.
2. Open terminal in project folder.
3. Run:

```bash
python bmi_calculator.py
```

## Example

Enter your weight (kg): 70
Enter your height (m): 1.75
